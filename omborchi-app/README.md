# Omborchi — Ombor boshqaruv tizimi

**by KAFTIMDA**

## O'rnatish

```bash
cd omborchi-app
npm install
npx expo start
```

## Firebase sozlash

`src/config/firebase.js` faylida `YOUR_API_KEY` va boshqa qiymatlarni Firebase Console dan oling.
